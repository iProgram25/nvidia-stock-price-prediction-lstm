# NVIDIA Stock Price Prediction using LSTM

## Overview

This project uses a Long Short-Term Memory (LSTM) neural network built with PyTorch to predict NVIDIA's next closing stock price using historical market data.

## Features

- Historical stock data from Yahoo Finance
- Data preprocessing with MinMaxScaler
- LSTM model built using PyTorch
- Training and testing on historical data
- Actual vs Predicted stock price visualization
- Next-day closing price prediction
- RMSE evaluation

## Technologies

- Python
- PyTorch
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- yfinance
